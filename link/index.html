链接合集

用于收集一些自己会用到的链接。

上财相关

  • 上财门户：用于管理上财信息，可以导向与上财有关网站的一站式平台
  • 上财教学网：搭载了bb平台，用于管理课程信息，接收作业、通知、课件以及查询分数
  • 上财教学技术中心：提供各种支持技术，例如正版软件下载，DNS域名解析，VPN等服务
  • 上财图书馆：用于查询图书存放情况以及借阅情况，方便去图书馆借书

编程学习

  • CSDN：具有博客，学院，论坛等等板块，机佬的天地之一
  • 学做网站论坛：用于学习搭建网站（Html + CSS + Javascript 的相关知识）
  • 菜鸟教程：提供各类语言的学习，安装工具过程的教程等等
  • 鱼C论坛：知名“互谅网”老师小甲鱼的的论坛
  • Stack overflow：用于查找错误和获取解决方法
  • Qt社区：用于了解和更好的使用编程软件Qt

博客相关

  • melody博客配置文档：本博客搭建采用的主题，需要查找配置在这里查找
  • 七牛云：图床网站，用于生成图片等外链
  • Cloudflare：用于设置网站DNS解析等

OJ平台

  • 洛谷：时下最受欢迎的在线OJ平台之一，有丰富的题解

日常收藏

  • 翻墙党论坛：用于获取 ssr / ss 等链接
  • 东莞桥牌网：包含桥牌论坛，理论、实战等一系列的讨论
  • 桥牌资料库：收藏了大量的桥牌教程以及攻略
  • 明日方舟ArkTool：部署在Github上的一个查询明日方舟干员信息的平台
  • 在线OCR：一款非常实用的在线OCR网站
